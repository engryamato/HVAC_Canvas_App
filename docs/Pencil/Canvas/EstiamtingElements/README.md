# Estimating Elements Design

Documentation for the visual representation and properties of estimating elements on the canvas.

## Categories

- **Accessories**: Dampers, sensors, and smaller components.
- **Equipments**: AHUs, Chillers, and major mechanical units.
- **Ducts**: Rectangular, Round, and Flex duct designs.
- **Fittings**: Elbows, Tees, and transitions.
