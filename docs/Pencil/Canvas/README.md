# Canvas Design Assets

This folder is dedicated to wireframes and design assets for the main application Canvas.

## Contents

- **BOM**: Bill of Materials design documentation.
- **EstiamtingElements**: Design for various physical elements (Ducts, Fittings, etc.).
- **ProjectProperties**: Layouts for project configuration and metadata settings.
