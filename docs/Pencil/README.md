# Pencil Design Assets

This directory contains design assets and wireframes created with Pencil Project for the HVAC Canvas Application.

## Overview

The design assets are organized into categories corresponding to different parts of the application:

- **Canvas**: Wireframes for the main drawing canvas, components, and tools.
- **Dashboard**: Wireframes for the project dashboard and management interface.

## Design Files

The master Pencil project files are located in the project root:

- [Improved Canvas.pen](../../Improved%20Canvas.pen): Main canvas and toolset design.
- [Improved Dashboard and Canvas](../../Improved%20Dashboard%20and%20Canvas): Dashboard and integrated canvas views.
- [hvac-wireframe.pen](../../hvac-wireframe.pen): Original wireframe designs.
